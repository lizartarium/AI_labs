В ходе лабораторной работы я решила задачу классификации изображений с использованием нейронных сетей.

Я загрузила и подготовила данные: провела разведочный анализ для оценки дисбаланса классов, выполнила предобработку изображений (приведение к единому размеру, нормализацию) и применила аугментации для улучшения обобщающей способности модели. Для обучения я использовала технику трансферного обучения на предобученной архитектуре (ResNet18/EfficientNetB0), адаптировав её под количество классов в датасете.

После обучения я провела комплексную оценку модели, рассчитав метрики Accuracy, Precision, Recall и F1-score, построила матрицу ошибок и ROC-кривые. Анализ результатов показал, какие классы модель путает чаще всего, и позволил выявить признаки переобучения. Визуализация предсказаний на выборке изображений наглядно подтвердила сильные и слабые стороны модели.

Работа позволила мне получить практический опыт построения полного пайплайна для компьютерного зрения — от анализа данных до интерпретации результатов предсказаний.
